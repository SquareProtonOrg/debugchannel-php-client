didCache()

didCache(string $id) : bool

Resolve cache request

If the ID is not present in the cache, then a new cache entry is created for the given ID, and string offsets are captured until cacheLock is called

This method must return TRUE if the ID exists in the cache, and append the cached item to the output, FALSE otherwise.
