timeFunc()

timeFunc(int $iterations, callable $function,  $output) : double

Executes a function the given number of times and returns the elapsed time.

Keep in mind that the returned time includes function call overhead (including microtime calls) x iteration count. This is why this is better suited for determining which of two or more functions is the fastest, rather than finding out how fast is a single function.

Parameters

int $iterations

Number of times the function will be executed
callable $function

timer()

timer(int $id, int $precision) : void|double

Timer utility

First call of this function will start the timer. The second call will stop the timer and return the elapsed time since the timer started.

Multiple timers can be controlled simultaneously by specifying a timer ID.

Parameters

int $id

Timer ID, optional
int $precision

Precision of the result, optional

Returns

void|double —

Elapsed time, or void if the timer was just started
